# Thư mục hình ảnh cho hệ thống quản trị bất động sản

Thư mục này chứa các hình ảnh sử dụng trong hệ thống gồm:

1. **logo.png** - Logo của hệ thống
2. **avatar.jpg** - Ảnh đại diện người dùng mặc định
3. **avatar1.jpg, avatar2.jpg,...** - Ảnh đại diện mẫu cho người dùng
4. **property1.jpg, property2.jpg,...** - Ảnh mẫu các bất động sản

Trong môi trường thực tế, hệ thống sẽ lưu trữ và quản lý hình ảnh tải lên từ người dùng.

Để tạo giao diện đẹp hơn, vui lòng thêm các hình ảnh thực tế vào thư mục này.
